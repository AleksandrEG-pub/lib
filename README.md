### Basic crud Library application. 

Contains only scripts with manual launch. 
No api exist.

Installation:
```
pip install --upgrade pip
pip install -e .
```

Database(pg postgres:17.5) is located in docker. To launch database run
```
./start-database.sh
```

Configurations
```
./.env
```

can be launched with ./src/library/__main

all operations are collected in ./src/library/lib.py

For convenience data for application can be generated by:
```
./src/library/service/data-generator.py
```
Call to generation present, but commented in 'lib.py'
